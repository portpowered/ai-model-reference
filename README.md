# Learn Language Models

[![CI](https://github.com/portpowered/ai-model-reference/actions/workflows/ci.yml/badge.svg)](https://github.com/portpowered/ai-model-reference/actions/workflows/ci.yml)

A static AI model reference site for large language model components, concepts,
papers, training regimes, systems ideas, and architecture variants.

visit now at https://portpowered.github.io/ai-model-reference/!

# feature set
1. website page search
2. documentation on various AI models, glossary, modules, papers, etc
3. tag based annotations of resources, i.e. various attention variants, forms of encoders, etc. 
4. graph presentations of modules
5. latency formula formatting
6. localization support for various languages
7. inline reference links to various associated pages
8. quick guides for fast learning of various topics such as transformer basics, model optimization
9. abstracted blog posts
10. timeline views
11. scale free graph view explorer of the documents
12. fancy graphs and what not showing the relationship between resources. 
13. various references for customers to look up as primary sources
